## Template
The lectures in this training material shall be formated using the templates in folder ```templates\```. Copy the files:

- TemplateLecture.ipynb (rename to suitable name)
- rise.json
- rise.css
- image1.png
- image2.jpg
- image5.jpg
 
To your new working folder of the lecture.
